Getting Something Working!

Oct 6th, 2010

Whenever I start on a project I always find it nice to get something working as soon as posible. Below is a screenshot of an android emulator which I’m accessing using a browser, which is definitely a nice first step and something to kick on from. Particularly pleasing was how responsive the emulator is, and how quickly it boots up. (Much faster than on my 2010 Macbook Pro.

The next steps are to work on having an android emulator boot and load an application as soon as a user logs in, as well as supporting multiple users at once.
