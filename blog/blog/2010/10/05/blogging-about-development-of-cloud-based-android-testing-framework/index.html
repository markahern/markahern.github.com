(Blogging About) Development of Cloud Based Android Testing Framework

Oct 5th, 2010

For my final year project I am working on creating an environment which can be used to easily test android applications via a web application on which the android emulator along with the application to be tested has already been set up and is accessible via a web browser. Here is the full project specification.

I will be blogging about my work in order to catalogue my efforts and also it may serve as a resource for people who come across problems that I solve in the course of the project.

I’ve already gotten started and in between lectures and other college assignments I have been working on getting an android emulator up and running on an Ubuntu Machine Image running on Amazon’s Elastic Compute Cloud.Â I hope to be able to access the emulator using x11vnc and guacamole. As it stands I am having difficulty getting x11vnc working, and guacamole is throwing me the following error:

Error connecting to server: net.sourceforge.guacamole.vnc.VNCException: Could not write to / read from socket (network error).

Not exactly much to report in the way of progress but I suppose it’s no harm to start blogging about whats happening from the very start!
